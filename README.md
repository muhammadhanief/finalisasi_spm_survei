## Menjalankan aplikasi

Instal Dependensi PHP

```bash
composer install
```

Instal Dependensi JavaScript

```bash
npm install
```

Jalankan Server Laravel

```bash
php artisan serve
```

Bangun dan Kompilasi Asset Frontend

```bash
npm run dev
```

(Opsional) Jalankan Migrasi Database

```bash
php artisan migrate
```

(Opsional) Jalankan Seeder Database

```bash
php artisan db:seed
```
